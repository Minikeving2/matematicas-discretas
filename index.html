Instrucciones rápidas
  • Modo Nodo: clic para añadir nodo.
  • Modo Arista: clic en dos nodos para crear arista.
  • Modo Mover: arrastra nodos.
  • Modo Borrar: clic en nodo o arista para eliminar.

Representaciones

1) Lista de adyacencia: Para cada nodo se listan sus vecinos.

2) Matriz de adyacencia: Tabla NxN que indica si existe arista entre dos nodos.

3) Lista de aristas: Muestra cada arista explícitamente como (u, v). Cada arista tiene un ID único mostrado en el canvas.
